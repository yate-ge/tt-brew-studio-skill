# Canonical Case：小雷《老年人可玩城市》Discover 阶段协同（我们要实现的目标）

这是**权威 case**，skill 要按它来做。范围：Discover → Define（到 HMW 收束为止），后面的方案
设计暂不做。协作机制（专家如何加内容/批注、Widget 四类、命名与免责）见
[collaboration-model.md](collaboration-model.md)。

## Pre

小雷是设计研究方向的学生，初始题目 **老年人可玩城市**——有吸引力但易空泛（"可玩"是什么？
是娱乐活动吗？从设施/空间/社交/体验出发？）。她进入协同画板描述兴趣：

> 我想研究老年人可玩城市，但现在还在前期探索阶段。我希望先整理已有资料、观察和访谈，再看看
> 这个方向有没有更清晰的研究切口。

Skill 判断她处于 **Discover 阶段**，目标不是马上出方案，而是打开问题、组织材料、发现线索、
逐步形成有研究价值的机会点。**8 位专家进入同一张画布持续协同**（不是各写一段点评）：

| 专家 | 领域 |
| --- | --- |
| 马谨 | 设计研究方法论 |
| 刘洋 | 城市数据驱动设计 |
| 魏佛兰 | 生态参与式设计 |
| 吴端 | 空间与标识设计 |
| 孙效华 | 智能交互设计 |
| 娄永琪 | 社会创新与可持续设计 |
| 辛向阳 | 交互设计理论 |
| 王受之 | 设计史与设计方法论 |

## 叙事 1：专家为小雷生成 Discover 研究脚手架

中央保留**课题卡**（初始命题：老年人可玩城市 / 当前阶段：Discover / 目标：收集材料、识别现象、
发现研究机会点）。围绕它，每位专家生成各自的 **Frame**；**画布上不放专家头像或署名 chip**，
悬停或选中左侧专家坝里的专家时，运行时用连线指向该专家负责的 Frame。使用说明保存在
`vd_usage_note`，在归属连线高亮时作为轻量浮层呈现。

- **马谨 · 研究问题拆解**：把"可玩城市"拆成可观察问题（哪些自发活动？哪些带探索/游戏/社交/
  愉悦？哪些空间支持/压抑？"可玩"是老人自己的说法还是设计者想象？）→ 把模糊概念拆成现象。
- **刘洋 · 城市行为与数据观察**：行为轨迹与空间节点（出现时间/活动地点/停留时长/移动路径/
  高频节点/触发行为的空间条件）；强调与阴影、座椅、菜市场、公交站、公园入口、街角广场相关。
- **魏佛兰 · 参与式观察**：进入老人真实现场（她在做什么/为什么停留/和谁互动/是否主动参与/
  是否旁观/如何描述这里）→ 尊重微小经验，不把老人塞进设计师预设。
- **王受之 · 文献研究**：文献分类（积极老龄化/适老化城市/可玩城市/城市公共空间/老年人社会
  参与/游戏化与日常生活/社区营造）→ 进入设计史、城市研究、社会议题脉络。
- **吴端 · 空间线索观察**：边界/路径/入口/标识/可识别性（如何识别空间？哪些边界愿停留？哪些
  路径愿探索？哪些标识带来安全感？哪些节点形成社交聚集？）。
- **孙效华 + 辛向阳 · 可玩体验触点**（共创一个 Frame）：看见→靠近→停留→试探→加入→退出→
  再次回来 → 提示可玩体验是低压力、可旁观、可退出的轻参与。
- **娄永琪 · 社会关系网络**：熟人相遇/陌生人搭话/信息交换/代际互动/被需要感/社区归属感 →
  价值不只是让老人"玩起来"，而是让老人重新成为城市公共生活的参与者。

阶段结束后画布不再空白，而是一组**专家审美 + 研究方法搭起来的 Discover 脚手架**。核心价值
第一次显现：**专家把研究品味转化为可操作的画布结构**。

## 叙事 2：小雷填入材料，与专家共同分析文献关键词

小雷把前期材料填进各 Frame：参与式观察板块传社区公园/菜市场/街角/公交站的观察照片；访谈
板块填访谈摘录；行为观察板块标注出现时间/地点/路径；文献板块贴 20 篇文献链接。**Skill 能理解
每个 Frame 里的内容、与课题的关系、以及小雷的说明。**

小雷选中文献板块请求："帮我分析这 20 篇文献的关键词，我想知道讨论集中在哪些方向。"→ 画布
在**文献研究 Frame 内**生成一个**词云分析 Widget**（可视化型，见 collaboration-model §Widget）。
读取标题/摘要/关键词，给出高频词（示例：active aging / public space / social participation /
age-friendly city / walkability / community / playfulness）。

词云旁专家**共创解释**：
- 王受之：主流讨论集中在"适老/健康/可达性/公共空间"，"playfulness/可玩性"频率较低——小雷的
  题目有机会补一个新研究视角。
- 马谨：高频词不等于研究方向，要看见高频词之间**缺失的连接**——"老年人"和"公共空间"常通过
  安全/健康/步行连接，较少通过游戏/探索/愉悦/表达连接。

核心价值：**AI 把分析结果放回专家脚手架中，让数据、文献、观察和专家判断发生关系。**

## 叙事 3：专家对整理好的 Discover 材料批注和建议（亮点）

小雷请求："我已按模板整理完前期信息，请各位专家从自己角度评论，告诉我问题、遗漏和下一步。"
→ 专家进入**评论模式**。专家意见统一进入**左侧专家栏**：专家头像显示意见数量，点头像展开该专家的
意见流；悬停或选中某条意见时，用连线指向具体画布内容，并高亮目标。

- **重大调整**：马谨在"研究问题描述"旁：你还在用"老年人需要更多可玩空间"描述问题，
  太早。Discover 应先回答：老人的哪些日常行为已经具有可玩性？哪些城市条件让它发生？
  娄永琪在社会关系 Frame：老人还像被观察对象，要补他们如何组织活动、维护关系、传递信息，
  否则回到"为老人设计"而非"和老人共同理解城市"。
- **小修建议**：刘洋在行为路径旁：记了在哪活动但没记时间节奏，补早/午/晚三段差异。
  吴端在空间照片旁：注意街角/树荫/台阶/栏杆/入口这些边缘空间——很多可玩性发生在城市边界处。
- **补充参考 / 方法**：魏佛兰在访谈区：建议补问"你为什么喜欢来这里""这里有什么意思"，
  不要直接问"你觉得好玩吗"（易让老人进入设计师预设）。孙效华在触点 Frame：记了做什么但没记
  如何进入活动，观察"从旁观到加入"的过程。

这些批注让小雷意识到材料够多但研究品味不够精确，学会继续追问（为什么愿停留？什么空间让人愿
观察别人？什么活动允许低压力参与？如何从旁观者变参与者？城市如何让老人觉得自己有能力、有关系、
有位置？）。核心价值：**专家的隐性审美通过左侧专家意见流和目标连线变得可追溯——哪个判断来自哪位专家、基于什么
视角、要求她如何修正观察。**

## 叙事 4：小雷回应批注，完成 Discover 收束，生成 HMW

小雷按建议改画布：补早/午/晚三段观察；把"你觉得好玩吗"改成"你为什么愿意来这里""最有意思的
时刻是什么"；在照片上标出树荫/栏杆/台阶/入口/小卖部旁停留点；把老人从旁观到加入广场舞/棋局/
聊天圈画成小流程图。

**每回应一条建议，该意见在左侧对应专家记录里标为已回应 / 已解决，不在主画布额外叠 UI。**
点马谨头像可见：原始批注 / 小雷的修改 / 修改时间 / 专家二次反馈。
点刘洋头像见所有行为路径/时间节奏/空间节点建议。点娄永琪头像见老人主体性与社会关系的完整
讨论记录。→ **整个协同过程可追溯，形成持续演化的研究过程档案。**

主要批注回应后，小雷切到**全局(zoom out)视角**，画布自动显示 Discover 整理结果（已完成 Frame /
已回应批注 / 待补材料 / 高频关键词 / 重要洞察 / 专家共同认可的机会）。她请求："基于我完成的
Discover 材料，帮我收束研究发现，生成一组 HMW。"→ 画布生成 **HMW 生成 Widget**（轻量输入+请求型），
调用全部材料（照片/访谈/词云/路径/批注/修改记录），专家共同产出 HMW，例如：

1. 如何让城市公共空间不仅对老人安全可达，也让他们产生探索和再发现的兴趣？
2. 如何支持老人在不被迫参与的情况下，从旁观、停留逐渐进入轻量社交？
3. 如何把街角、树荫、台阶、入口等边缘空间，转化为老人愿意停留和互动的可玩节点？
4. 如何设计低压力、可退出、可重复参与的城市活动机制，让老人保持主动性？
5. 如何让老人的经验、记忆和地方知识成为城市可玩性的一部分，而不只是被服务对象？
6. 如何通过路径、标识和微型任务，帮助老人在熟悉社区重新获得探索感？
7. 如何让老人的日常社交从偶然发生，转化为被城市空间温和支持的公共生活？

HMW Widget 旁专家给收束性判断：**老年人可玩城市的核心，是重新理解他们在城市中的日常参与方式——
可玩性来自安全之后的探索、熟悉之中的变化、旁观到加入的弹性，以及被看见、被需要、能表达的城市
关系。** 至此 Discover 完成：从模糊题目出发，形成一组有证据、有专家判断、有研究审美支撑的
设计机会点。

## Case 的核心思想

在共享画布上组织一场多专家协同：专家通过 **Frame** 引导学生如何看问题，通过 **Widget** 和学生
共同分析材料，通过**批注**让学生学习专业判断，再通过**可追溯的迭代记录**帮助学生完成阶段收束。
最终学生获得一套逐步内化的设计研究判断力。
